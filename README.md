# Checkers Game (Python & Pygame)

A 2D Checkers game implemented in Python using the Pygame library. This project features a graphical interface with a score tracker for Red and Black players.

## Prerequisites

Before running this project, ensure you have the following installed on your system:
* **Python 3.10+**
* **python3-venv** (Required for Linux users to create virtual environments)

To install the venv tool on Ubuntu/Debian:
```bash
sudo apt update
sudo apt install python3-venv
source .venv/bin/activate
pip install pygame
pip install pygame
File Structure

    main.py: Entry point for the game.

    board.py: Contains the logic for the checkers board and movement.

    game.py: Handles game state and turn management.

    piece.py: Defines the properties of individual checkers pieces.

    assets/: Folder containing images and sounds (if applicable).

    constants.py: Stores global variables like colors, screen size, and dimensions.
